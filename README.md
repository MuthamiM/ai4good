**FinWise AI** is a comprehensive financial inclusion platform designed to empower underserved communities with personalized, AI-driven financial insights, budgeting tools, and loan eligibility verification.

##  Live Application
**Live Demo:** [https://mussa21.pythonanywhere.com](https://mussa21.pythonanywhere.com)

---

##  Key Features

### 1. Financial Core Modules
- **Budget Analyzer**: Smart calculations for 50/30/20 budget rules based on income.
- **Loan Eligibility Checker**: Instant feedback on loan readiness based on user risk profiles.
- **Expense Categorizer**: Visualizing spending habits to identify saving opportunities.
- **Savings Advisor**: AI-driven strategies for long-term financial stability.

### 2. AI & ML Integration
- **Financial Chatbot**: A real-time AI assistant for answering complex financial questions and providing personalized advice.
- **Data Heuristics**: Algorithms to categorize and optimize budgets for social good.

### 3. Professional UI/UX
- **Glassmorphism Design**: Modern, clean, and intuitive dashboard.
- **Responsive Layout**: Seamlessly accessible via desktop or mobile devices.
- **Dynamic Animations**: Enhanced user experience for financial tracking.

##  Technology Stack
- **Backend:** Python + Flask
- **Database:** SQLite3
- **Frontend:** HTML5, CSS3, JavaScript
- **Auth:** Clerk / Global Auth Integration
